# Olivier app icon

`olivier.svg` (and the rasters generated from it by `scripts/gen-icons.sh`) is an original "birdsong
in colour" motif created for this project — a songbird with coloured notes, evoking Olivier
Messiaen's birdsong and colour synesthesia.

Released under the Creative Commons CC0 1.0 Universal Public Domain Dedication (no rights reserved).
No third-party assets are used.
